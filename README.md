# 🏋️ FitFlix

Sistema web para visualização de exercícios físicos com interface inspirada em plataformas de streaming (estilo Netflix).

---

## 🚀 Funcionalidades

- 📂 Organização por grupos musculares (Peito, Costas, etc)
- 🎬 Visualização interativa dos exercícios
- 🖼️ Imagens e GIFs demonstrativos
- 📋 Modal com variações de exercícios
- 🧠 Interface pensada para facilitar a experiência do usuário

---

## 🛠️ Tecnologias utilizadas

- Python (Django)
- HTML5
- CSS3
- JavaScript
- MySQL
